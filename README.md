# Podcast Clips

A full Android podcast client with precise local clipping and optional low-cost hosted transcription.

The project is bootstrapped and validated entirely with GitHub Actions. See `DESIGN.md` for the product design constitution and `UPSTREAM.md` for snapshot provenance.

## Story 1 status

The repository begins with a trusted control plane. A pinned GPL-3.0 upstream candidate is imported as data into a quarantined candidate branch, then executed only in separate no-secret, no-token validation jobs. No upstream workflow or hook is activated during import.
