# Product dependencies

The foundation uses the approved AndroidX, Kotlin, Media3, Room, WorkManager,
OkHttp, kotlinx, Coil, and test-only dependency pins. All runtime components are
Apache-2.0, MIT, BSD, or similarly GPL-3.0-compatible. JUnit is test-only.
No Groq/OpenAI SDK, analytics SDK, advertising SDK, DI framework, or second HTTP
stack is present.
